# Cursor skills (project)

This repo vendors **[code-review-skill](https://github.com/awesome-skills/code-review-skill)** under `code-review-skill/`. Cursor discovers skills in `.cursor/skills/` automatically.

**Invoke:** In Agent chat, use `/code-review-excellence` (see `SKILL.md` frontmatter `name`) or `@` to attach the skill. Example prompt:

> Use the code-review skill to review the changes on this branch; load `reference/typescript.md` for TS/React code.

See [Cursor Skills docs](https://cursor.com/docs/skills).
